## 验收标准

所有页面必须满足以下条件：

1. **视觉一致性**: 
   - 与原型进行逐像素对比，布局、间距、阴影、字体、图标、背景等高度一致
   - 响应式断点与原型匹配

2. **Material Symbols 图标**:
   - 所有 Material Symbols 图标必须正确渲染为图标字体
   - 不允许显示为文本内容（如 "smart_toy"、"person" 等）

3. **功能保留**:
   - 所有现有的业务逻辑和 API 调用必须正常工作
   - 表单提交、数据加载、用户交互等功能不受影响

4. **性能和体验**:
   - 页面加载速度正常
   - 动画过渡流畅
   - 无控制台错误或警告

## 测试方法

1. 启动开发服务器: `npm run dev`
2. 访问每个路由
3. 使用浏览器开发工具对比原型 HTML 和实际渲染结果
4. 检查 Material Symbols 是否正确显示
5. 测试所有交互功能是否正常
